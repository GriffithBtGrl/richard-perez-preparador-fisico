# Richard Perez Bran – Sitio Web Profesional

Sitio web profesional para **Richard Perez Bran**, preparador físico y boxeador.
El objetivo del proyecto es presentar sus servicios, experiencia y facilitar el contacto con potenciales clientes interesados en entrenamiento y planes online.

## 🥊 Sobre el proyecto

- Sitio web one-page
- Enfoque en marca personal y conversión
- Diseño oscuro, sobrio y deportivo
- Optimizado para dispositivos móviles
- Contenido orientado a entrenamiento y nutrición online

## 🛠 Tecnologías utilizadas

- HTML5 semántico
- CSS3 (responsive design + animaciones)
- JavaScript vanilla
- SEO básico y Open Graph
- Imágenes optimizadas en formato WebP

## 📱 Funcionalidades

- Navegación responsive
- Botón flotante de contacto por WhatsApp
- Enlaces directos a correo electrónico e Instagram
- Hero optimizado para desktop y mobile
- Animaciones suaves al hacer scroll
